# Software Camera Coverage Check

## Problem
We want to build a software camera by combining multiple hardware cameras.  
Each hardware camera supports a specific range of subject distances and light levels.

The goal is to verify whether a given set of hardware cameras can fully cover the required distance and light level ranges of the software camera.

---

## Solution
- Implemented using plain JavaScript
- No UI and no third-party libraries
- Focused purely on logical correctness and clarity
- Implemented in a single file

---

## Tech Stack
- JavaScript (ES6)
- Node.js

---

## Run Program
```bash
node camera.js 
```
